The meta layer

How the collaboration stays correct

Behind the curriculum sits a second system whose only job is to keep the first one honest: it mines its own mistakes, ships fixes, and then measures whether they worked — showing the misses as plainly as the wins. It corrects the collaboration, never the teacher's judgment.

The mine → ship → measure flywheel

Recurring mistakes are treated as data. Each turn of the wheel finds a failure shape, ships a fix for it, and re-counts how often it still happens — so an intervention that backfired is caught by the same instrument that praises the ones that worked.

Why the receipts are rates, not spend

Every receipt below is an occurrence rate — how often the targeted mistake still shows up per window of work. We deliberately do not publish effort or spend deltas: the way that was measured shifted mid-window, so those numbers would not be comparable across the baseline. The rate is. Feedback here is informational, not a score.

The invariant registry, by tier

Some corrections must hold across every workstream, not just the one they arose in. They live in a frozen registry, each with an enforcement tier — from a prompt-time reminder up to a tool-layer block that no rewording can dodge.

The hook immune system

Rules on their own rot — unenforced, they get forgotten. So the load-bearing ones are wired to lifecycle hooks that fire automatically as the work happens: before a tool runs, after it runs, when a turn ends. They are the reflexes that catch a slip before it ships.

Recall — the memory that arrives on time

The hardest failure to catch is forgetting what was already built or already decided. The built-in session memory was not enough for that — so the collaboration grew its own layer on top: an inventory of everything the work has produced, pushed into the conversation at the moment a prompt lands, so a past decision shows up before it is re-litigated, not after. And because an unmeasured memory would just be more noise, the layer audits its own usefulness.

Why the low number renders

Every injection is joined back to its session's transcript and asked one question: was this actually consulted? Only a demonstrable touch counts, so advice that was simply read in place scores zero — the rate is a floor, not a grade. What matters is that the memory acts on its own audit: entries that keep surfacing without ever being used are pruned or down-ranked automatically. A memory that only praised itself would be worthless here.

The collaboration audits itself, out loud. The flywheel shows its regressions next to its wins; the registry escalates a correction until it cannot be dodged; the hooks fire on their own; recall brings the past forward in time. None of it grades a student — it keeps the machinery honest so the teacher's judgment is the one that counts.
