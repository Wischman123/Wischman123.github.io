The dashed ring is the moat: 0 imports cross it, in either direction, re-proven on every build. The moat, gated

The assessment engine and the product plane — 2 island planes in this map — exchange 0 imports with the 5 fabric regions, in either direction. That absence is the ports-and-adapters invariant: dependencies point only inward, so the subject-free core never reaches up into the machinery that builds, checks, and tunes the curriculum. It is re-proven against the harvested graph on every build — a drawn moat with a failing gate behind it, not a caption.
