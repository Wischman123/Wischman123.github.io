Zoom out from the hexagon

The engine, drawn as a landscape

The assessment hexagon is one region of a larger machine: the meta-cognitive fabric that builds, checks, and keeps the curriculum honest. Positions and captions here are curated for reading; every drawn edge is a real import, recomputed from the two node groups against the harvested module graph. A hand-declared edge with no underlying import fails the drift gate.

Every arrow below is one of the 1,129 imports across a 422-module graph, recomputed from the two node groups it joins — 13 proven edges across 10 subsystems. Node GROUPS are dotted module names in the ONE canonical witness source (import_graph.json). Every listed module must exist in that graph's node set (presence gate); every edge's witness is the NON-EMPTY set of module-pair imports between the two groups (drift gate). catalog_manifest.json is NOT a second witness source.

Isolated by design
